# Audience Analyzer

Анализ целевой аудитории из чатов вебинаров. CSV → JTBD-анализ → HTML-дашборд.

## Быстрый старт

```bash
# Положить CSV в input/
cp "Выгрузка из чата.csv" input/chat_day1.csv

# Запустить анализ
python src/run.py input/chat_day1.csv

# Открыть результат
open output/dashboard.html
```

## Что на выходе

Интерактивный HTML-дашборд с вкладками:
- **Портреты ЦА** — 3 портрета с болями, желаниями, цитатами
- **Jobs to be Done** — зачем пришли (заработать, создать продукт, найти клиентов...)
- **Боли и страхи** — что мешает (не получится, дорого, сложно...)
- **Желаемые продукты** — что хотят создать (боты, приложения, сайты...)
- **Ниши** — кто в аудитории (IT, образование, строительство...)
- **Рекомендации** — как адаптировать оффер под данные
- **Структура оффера** — что говорить каждому сегменту

## Формат CSV

```
Время,id отправителя,Имя,id получателя,Сообщение
2026-03-09 18:15:32,user123,Алексей Иванов,chat456,Хочу создать бота для записи клиентов
```

## Два дня (сравнение)

```bash
python src/run.py input/chat_day1.csv input/chat_day2.csv
```

Добавляется секция сравнения: что изменилось между днями, какие jobs выросли, новые темы.
